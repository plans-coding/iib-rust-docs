# Translations

## App translation

If the app is not available in your language you can easily translate it by placing a file in the folder **personal/** and change the value in settings table `bewxx_Settings`.

Read more under [Settings](settings.html).

## Continent and Country translations
The continent and countries can be translated, read more [Table: Continent Countries](trip-syntax.html#table-continent-countries).

## Trip Domain translation
The trip domains can be translated, read more [Table: Trip Domains](trip-syntax.html#table-trip-domains).
